== Description ==
Encode strings or words that are sensitive, vulgar, or violate guidelines within the comments.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload zip file in the 'Plugins' menu. Or Upload manual folder `safe-comment-content` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter the word you want to encode, one word per line.
